# AirSenseAI

AI-Based Real-Time Pollution Prediction and Alert System

## Overview

AirSenseAI predicts future Air Quality Index (AQI) levels using machine learning and environmental data.

The system focuses on Karnataka cities with AQI monitoring stations and provides:

* Real-time AQI information
* 24-hour AQI forecasts
* AQI trend forecasts
* Pollution risk indicators
* Health recommendations

## Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* OpenWeather API
* CPCB AQI Data
* WAQI API

## Machine Learning Model

Random Forest Regressor

## Responsible AI Principles

* Uses trusted public data
* Does not collect personal information
* Provides advisory recommendations only
* Displays prediction limitations and data sources
